# RPorn - GitHub Pages Mirror

This is a GitHub Pages mirror of the RPorn Blogger blog.

## Files Included

- **index.html** - Main homepage (this is the file GitHub Pages will serve)
- **favicon.ico** - Blog favicon
- **Images** - Blog images and media files
- **feed.xml** - Blog feed (Atom format)
- **metadata.json** - Blog metadata
- **settings.csv** - Blog settings
- **theme-backup.html** - Original Blogger theme backup

## How to Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., `username.github.io` or `rporn`)
2. Upload all files from this folder to the repository root
3. Go to repository Settings → Pages → Source and select "Deploy from a branch"
4. Select the branch (usually `main` or `master`) and save
5. Wait 1-2 minutes for GitHub to build your site
6. Your site will be available at `https://username.github.io` or `https://username.github.io/rporn`

## Important Notes

- The **index.html** file MUST be in the root directory of your repository
- All image files should be in the same directory as index.html
- GitHub Pages takes 1-2 minutes to deploy after changes
- Clear your browser cache if changes don't appear immediately
